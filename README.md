# GreenEdge Mowing Website

Source files for the GreenEdge Mowing website, built with HTML, CSS and Tailwind CSS.

## Structure

- `index.html` - Main marketing site
- `greenedge-crew.html` - Crew portal
- `sign up/index.html` - Crew signup form
- `404.html` - Custom not-found page
- `GREENEDGE LOGO.png` - Brand logo

## Deployment

This site is deployed via GitHub Pages.
